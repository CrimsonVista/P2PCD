- To install sumo do following
git clone https://github.com/eclipse/sumo.git
git checkout v0_32_0
sudo apt-get install cmake python g++ libxerces-c-dev libfox-1.6-dev libgdal-dev libproj-dev libgl2ps-dev swig
cd <SUMO_DIR> # please insert the correct directory name here
export SUMO_HOME="$PWD"
mkdir build/cmake-build && cd build/cmake-build
cmake ../..
make -j$(nproc)

- To run the simulations with openssl:
include ssl library
link libssl and libcrypto to the src folder like in <veins_folder>/src
ln -s /lib/x86_64-linux-gnu/libcrypto.so.1.0.0 libcrypto.so
ln -s /lib/x86_64-linux-gnu/libssl.so.1.0.0 libcrypto.so

open configurations -> Main -> More>> -> Dynamic libraries -> Browse -> select libcrypto.so libssl.o libveins.so -> Apply

Run Final Step in the tutorial
https://veins.car2x.org/tutorial/
